#  Snake Game in Python (Turtle & Pygame)

A classic Snake game built in Python using two approaches:
- **Turtle** (beginner-friendly)
- **Pygame** (game-engine style)

---

##  Features
- Snake movement with arrow keys
- Food spawning and snake growth
- Wall collision
- Self-collision
- Score system

---

##  Turtle Version
**Requirements:** Python 3.x  
**Run:**
```bash
python snake_game_turtle.py
